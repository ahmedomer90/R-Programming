# Scripts

This directory contains all R scripts used in the reconstruction of my MSc Cancer dissertation.

The scripts will be organised in the order of the computational workflow.

## Planned workflow

1. Data acquisition
2. Data import
3. Data quality control
4. Data preprocessing
5. Copy Number Alteration (CNA) analysis
6. Statistical analyses
7. Data visualisation
8. Export of results

Each script will be numbered to reflect the order in which it should be executed.
